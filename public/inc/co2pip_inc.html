Welcome to the Phanerozoic CO2 Proxy Integration Project (CO2PIP)!

This is the ongoing phase of the paleo-CO2 project and aims to (1) expand the paleo-CO2 proxy data repository to the entire Phanerozoic, (2) modernize existing CO2 proxy records, (3) develop forward proxy system models for the paleosol, leaf-based, phytoplankton, and boron isotope proxies, and (4) statistically integrate proxy models and data to generate a robust, quantitative reconstruction of Phanerozoic atmospheric CO2. In addition, CO2PIP develops (5) a set of 2-D and 3-D tools for users to experience regional CO2/climate connections, and (6) paleo-CO2 teaching modules tailored to undergraduate and graduate students.

 
This plot shows paleo-CO2 estimates published to date. Following vetting criteria established for the Cenozoic, gray symbols on this plot are considered either semi-quantitative or in some cases unreliable in their current form. Modernizing these data is the main target of CO2PIP.
